Web Emotion Detection

Deteksi emosi dari banyak wajah langsung di browser.

Aplikasi ini mengubah skrip kamera Python menjadi web app. Buka halaman ini, izinkan kamera, lalu sistem akan menandai setiap wajah dan membaca emosi seperti senang, sedih, marah, takut, dan lainnya.
